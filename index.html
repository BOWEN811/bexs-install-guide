BEXS 技能包 · 安裝教學

3 分鐘把技能裝進你的 Claude Code，跟著做就會 ✋

💡 開始前確認：你電腦已經裝好 Claude Code（在終端機打 claude 有反應）。沒有的話先去裝。

1拿到技能包

老師會給你「下載連結」或一個 ZIP 壓縮檔。

• 拿到 連結：點右上角綠色 Code → Download ZIP

• 拿到 ZIP 檔：直接用。

2解壓縮

雙擊那個 ZIP 解壓縮，會得到一個資料夾，裡面有個 skills 子資料夾，裝著一個個技能。

3打開「終端機」

Mac：按 ⌘ + 空白鍵 → 打「終端機 / Terminal」→ Enter 打開。

4把技能複製進 Claude

先 cd 進你剛解壓縮的資料夾（把資料夾拖進終端機最快），然後貼上下面指令：

裝單一個技能（例：廣告導演）：

cp -r skills/ad-director ~/.claude/skills/

一次裝全部：

for d in skills/*/; do cp -r "$d" ~/.claude/skills/; done
📁 ~/.claude/skills/ 就是 Claude 放技能的地方，貼指令會自動放對位置，不用自己找。

5重開 Claude Code

關掉 Claude Code 再重新打開（讓它讀到新技能）。

6開始用！

兩種用法：

• 白話講：直接說你要做什麼（例：「幫我把這支影片切成短片」），對應技能會自動跳出來。

• 打技能名：在指令裡輸入英文技能名，例如 ad-director。

✅ 看到技能會動作，就裝成功了！

❓ 常見問題

Q：裝了叫不出來？
確認你複製的是 skills/ 裡面的技能資料夾，而且放到了 ~/.claude/skills/，然後重開 Claude Code。
Q：哪些技能要額外環境？
影音 / 生圖類技能（製片、生圖、生影片）需要對應工具（ComfyUI、Higgsfield 帳號、本地模型等）。文字 / 分析類（交易體檢、找客戶、網頁審核）裝了直接能用。
Q：每個技能在幹嘛？
進到該技能資料夾，打開裡面的 SKILL.md，就有完整說明。或看技能包 README 的對照表。
Q：我不是用 Mac？
Windows / Linux 一樣，把技能資料夾複製到 ~/.claude/skills/（家目錄下的 .claude/skills）即可。
