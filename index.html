BEXS 技能包 · 安裝 + 技能介紹

3 分鐘裝好，26 個 AI 技能讓 Claude Code 幫你做事 ✋

💡 開始前確認：你電腦已裝好 Claude Code（終端機打 claude 有反應）。沒有的話先去裝。

📥 安裝步驟

1拿到技能包

老師會給你「下載連結」或一個 ZIP 壓縮檔。

• 拿到 連結：點右上角綠色 Code → Download ZIP

• 拿到 ZIP：直接用。

2解壓縮

雙擊 ZIP 解壓縮，會得到一個資料夾，裡面有 skills 子資料夾，裝著一個個技能。

3打開「終端機」

Mac：按 ⌘ + 空白鍵 → 打「終端機 / Terminal」→ Enter。

4把技能複製進 Claude

先 cd 進你解壓縮的資料夾（把資料夾拖進終端機最快），再貼指令：

裝單一個（例：廣告導演）：

cp -r skills/ad-director ~/.claude/skills/

一次裝全部：

for d in skills/*/; do cp -r "$d" ~/.claude/skills/; done
📁 ~/.claude/skills/ 是 Claude 放技能的地方，貼指令會自動放對位置。

5重開 Claude Code

關掉再重開，讓它讀到新技能。

6開始用！

• 白話講：直接說你要做什麼，對應技能自動跳出來。

• 打技能名：輸入英文技能名，例如 ad-director。

✅ 看到技能動作，就裝成功了！

🧰 26 個技能完整介紹

每個技能能幹嘛 + 怎麼開口叫它。安裝/呼叫一律用英文名。

⭐ 招牌創作 / 工具技能（15）
🎬 廣告導演 ad-director
像請了一位資深廣告導演，從寫劇本、分鏡、運鏡、生成畫面、配旁白、配樂、剪輯到上字卡，整支商業廣告片一條龍幫你做到「可上市」等級，每一步都過審核才放行。
💬 怎麼說：「幫我拍一支廣告」「做支精品錶廣告片」「重剪這支片」
🎞️ AI影片製片 ai-film-studio
一句話劇本就能變成一支片：自動生分鏡、建立角色/場景一致性設定、生關鍵畫面、生影片、剪接成片。適合做故事片、MV、宣傳片。
💬 怎麼說：「把這個故事做成影片」「做一支片」「一句話變影片」
✂️ 長片切短影音 video-shorts
把一支長影片自動切成一批適合 IG / TikTok / YouTube Shorts 的 9:16 直式短片，自動抓重點、裁直式、加動態字幕。特別適合知識型乾貨。
💬 怎麼說：「把這支影片切成短片」「做幾支 Reels」「切重點片段」
📺 動態廣告 motion-ad
做 15 秒「不像 AI」的精緻動態設計廣告，適合接案作品集或交付客戶，可一次量產多種顏色/產業變體。
💬 怎麼說：「做一支動態廣告」「做支 15 秒產品廣告」「量產廣告變體」
🎙️ YouTube轉Podcast yt-to-podcast
丟一個 YouTube 連結，自動下載字幕、翻成中文、整理成 S/A/B 級重點報表，再做成 NotebookLM 的 Podcast 音檔。
💬 怎麼說：「把這影片做成 Podcast」「整理這支影片重點」（貼 YT 連結）
🖼️ AI生圖生片 higgsfield-generate
用 Higgsfield 生高品質圖、影片、圖生影片、真人代言 UGC 帶貨廣告。要付費帳號但品質高。
💬 怎麼說：「生一張圖」「做支廣告」「動畫這張照片」「做 UGC 影片」
👤 數位分身訓練 higgsfield-soul-id
訓練一個專屬你臉的模型，之後生圖生片都是「一致的你」，可做個人 IP、虛擬代言、形象照。
💬 怎麼說：「訓練我的臉」「做我的數位分身」「建一個我的 avatar」
📸 產品攝影 higgsfield-product-photoshoot
把一張普通產品照變成品牌級棚拍圖、情境生活照、廣告素材、虛擬試穿，免找攝影棚。
💬 怎麼說：「把這產品拍漂亮」「做產品情境圖」「做廣告素材」
🛒 電商商品圖卡 higgsfield-marketplace-cards
做電商上架用的商品主圖、細節說明卡、A+ 內容模組，符合蝦皮 / Amazon 賣場規範。
💬 怎麼說：「做蝦皮上架圖」「做 Amazon 商品卡」「做 A+ 內容」
📊 交易單體檢 bexs-trade-audit
丟一張盤面圖或交易描述，用 BEXS 系統（時段紀律 / SOP / 止損 / 心態紀律 / 錯誤反指標）幫你打分、列出違紀清單、給修正建議。學員自助復盤神器。
💬 怎麼說：「幫我看這張圖」「這單可以進嗎」「復盤這筆交易」
🎯 找客戶名單 client-finder
自動從 Google Maps、Kickstarter、賣家平台找潛在客戶負責人的 email，過濾死帳號，輸出乾淨可寄送的開發名單。
💬 怎麼說：「幫我找客戶」「抓開發名單」「找接案對象」
🔥 煉丹爐模型微調 lora-forge
在自己電腦上微調本地 AI 模型（LoRA）的完整流程：收料 → 訓練 → 評估 → 部署，把模型養成你專屬的。進階玩家用。
💬 怎麼說：「微調模型」「煉丹」「養我自己的 AI」
🔍 網頁審核 web-audit-supervisor
實際打開你的網站跑十層全面體檢（RWD / SEO / 速度 / console / 死連結 / 表單…），產出問題清單 + 嚴重度 + 修復指令。當你的外部驗收 QA。
💬 怎麼說：「幫我審一下網站」「驗收這個網頁」「抓 bug」
⚡ 網頁效能體檢 web-perf
量你網站的 Core Web Vitals（LCP/INP/CLS）、Lighthouse 分數，找出拖慢載入的原因。
💬 怎麼說：「網站好慢」「測一下速度」「優化效能」
📥 抓Notion公開頁 notion-public-fetch
把 Notion 公開分享頁的內容免 token 抓下來轉成 markdown，含巢狀子頁面。整理課程 / 知識庫超好用。
💬 怎麼說：「抓這個 Notion 頁」（貼 notion 連結）

❓ 常見問題

Q：裝了叫不出來？
確認你複製的是 skills/ 裡的技能資料夾，放到 ~/.claude/skills/，然後重開 Claude Code。
Q：哪些技能要額外環境？
影音 / 生圖類（製片、生圖、生影片）需對應工具（ComfyUI、Higgsfield 帳號、本地模型）。文字 / 分析類（交易體檢、找客戶、網頁審核）裝了直接能用。
Q：每個技能更深入的說明在哪？
進到該技能資料夾，打開裡面的 SKILL.md，有完整用法。
Q：不是用 Mac？
Windows / Linux 一樣，把技能資料夾複製到家目錄下的 .claude/skills/ 即可。
