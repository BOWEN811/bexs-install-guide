BEXS 技能包 · 安裝 + 技能介紹

3 分鐘裝好，26 個 AI 技能讓 Claude Code 幫你做事 ✋

💡 開始前確認：你電腦已裝好 Claude Code（終端機打 claude 有反應）。沒有的話先去裝。

📥 安裝步驟

1拿到技能包

老師會給你「下載連結」或一個 ZIP 壓縮檔。

• 拿到 連結：點右上角綠色 Code → Download ZIP

• 拿到 ZIP：直接用。

2解壓縮

雙擊 ZIP 解壓縮，得到一個資料夾，裡面有 skills 子資料夾，裝著一個個技能。

3打開「終端機」

Mac：按 ⌘ + 空白鍵 → 打「終端機 / Terminal」→ Enter。

4把技能複製進 Claude

先 cd 進你解壓縮的資料夾（把資料夾拖進終端機最快），再貼指令：

裝單一個（例：廣告導演）：

cp -r skills/ad-director ~/.claude/skills/

一次裝全部：

for d in skills/*/; do cp -r "$d" ~/.claude/skills/; done
📁 ~/.claude/skills/ 是 Claude 放技能的地方，貼指令會自動放對位置。

5重開 Claude Code

關掉再重開，讓它讀到新技能。

6開始用！

• 白話講：直接說你要做什麼，對應技能自動跳出來。

• 打技能名：輸入英文技能名，例如 ad-director。

✅ 看到技能動作，就裝成功了！

⭐ 招牌創作 / 工具技能（15）
🎬 廣告導演 ad-director🔗 連結
像請了一位資深廣告導演（拍過精品錶/車/酒的等級），從寫劇本、分鏡、運鏡、生成畫面、配旁白、配樂、剪輯到上字卡，整支商業廣告片一條龍幫你做到「可上市」等級。每一步都過「上市檢核表」審核，不到位不出片。台詞由導演自己寫到位，不丟回給你。
💬 怎麼說：「幫我拍一支廣告」「做支精品錶廣告片」「重剪這支片」「配個旁白」
🎯 適合：要做精品/產品商業廣告、想要展出級品質、接案交付客戶。
🎞️ AI影片製片 ai-film-studio🔗 連結
一句話劇本就能變成一支片：自動寫分鏡、建立角色與場景的「一致性設定（World Bible）」、生關鍵畫面、生影片、剪接成片。本地免費出片或接付費高品質都行。
💬 怎麼說：「把這個故事做成影片」「做一支片」「一句話變影片」「做支 MV」
🎯 適合：YouTuber、把小說/故事做成動畫、宣傳片、形象片。
✂️ 長片切短影音 video-shorts🔗 連結
把一支長影片自動切成一批適合 IG / TikTok / YouTube Shorts 的 9:16 直式短片：自動轉錄、挑出完整重點段落、裁直式、加動態逐詞字幕。全本地、$0。
💬 怎麼說：「把這支影片切成短片」「做幾支 Reels」「切重點片段」
🎯 適合：有長片想經營短影音、知識型/財經教學創作者。

做 15 秒「不像 AI」的精緻動態設計廣告（motion design）。引導你填 logo、時長、風格、分鏡，自動生分鏡再出片，可一次量產多種顏色/產業/型號變體。

❓ 常見問題

Q：裝了叫不出來？
確認你複製的是 skills/ 裡的技能資料夾，放到 ~/.claude/skills/，然後重開 Claude Code。
Q：哪些技能要額外環境？
影音 / 生圖類（製片、生圖、生影片）需對應工具（ComfyUI、Higgsfield 帳號、本地模型）。文字 / 分析類（交易體檢、找客戶、網頁審核）裝了直接能用。
Q：每個技能更深入的說明在哪？
進到該技能資料夾，打開裡面的 SKILL.md，有完整用法。
Q：怎麼把單一技能分享給別人？
點該技能右上角「🔗 連結」會複製它的專屬網址，貼給別人就能直接跳到那個技能。
Q：不是用 Mac？
Windows / Linux 一樣，把技能資料夾複製到家目錄下的 .claude/skills/ 即可。
